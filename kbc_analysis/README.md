# KBC Void Comparative Analysis

This directory contains the quantitative comparison between the BHC model and the KBC Void observational data (Keenan, Barger & Cowie 2013).

## Files

- `kbc_comparison.py` : Python code to generate the comparison figure
- `BHC_vs_LCDM_KBC_Void.pdf` : Vector graphic for publication
- `BHC_vs_LCDM_KBC_Void.png` : Bitmap image for web/social media
- `Supplementary_KBC_Void_Analysis.pdf` : Full supplementary document

## Key Results

- BHC model predicts density contrast δ ≈ 0.46 at z ≈ 0.055
- KBC Void observed δ = 0.46 ± 0.06
- ΛCDM excluded at 6.04σ confidence (Haslbauer et al. 2020)
